# IMDb Top 250 Movie Scraper

## Overview
This project is a Python-based web scraper that extracts information about the **Top 250 movies** from [IMDb](https://www.imdb.com/chart/top/). It collects details such as:

- Movie title  
- Release year  
- IMDb rating  
- Director and main cast (optional)  

The extracted data is stored in a **CSV file** for easy analysis and further use.

---

## Features

- Scrapes the latest IMDb Top 250 movies  
- Saves the data into a structured CSV file  
- Automatically handles pagination or scrolling (if applicable)  
- Lightweight and easy to run  

---

## Technologies Used

- **Python 3.x**  
- **Selenium** for web scraping  
- **Pandas** for data handling  
- **ChromeDriver** for browser automation
